Scratch 2.0 offline Installs (http://scratch.mit.edu/scratch2download/)

Open Scratch 2.0 offline, when you press the Shift key while you click on the File menu, the menu "Import experimental HTTP extension" shown

https://github.com/electronicadivertida/scratch2LittleBits/blob/master/scratch2LittleBits/scratch2LittleBits/doc/uso/scratch2Littebits1.png

Choose the scratch2LittleBitsDEF.json

https://github.com/electronicadivertida/scratch2LittleBits/blob/master/scratch2LittleBits/scratch2LittleBits/doc/uso/scratch2Littebits2.png

Now you can access the littleBit blocks in the "More Blocks" tab

https://github.com/electronicadivertida/scratch2LittleBits/blob/master/scratch2LittleBits/scratch2LittleBits/doc/uso/scratch2Littebits3.png

Now to work around, connect the Arduino to the computer module, load the program using the Arduino IDE. (Full instructions here: http://discuss.littlebits.cc/t/getting-started/109)

https://github.com/electronicadivertida/scratch2LittleBits/blob/master/scratch2LittleBits/scratch2LittleBits/arduino/littleBits_Scratch2/littleBits_Scratch2.ino


Execute with java bin/scratch2LittleBits.jar (This directory was not, sorry)

https://github.com/electronicadivertida/scratch2LittleBits/tree/master/scratch2LittleBits/scratch2LittleBits/bin
